# Яндекс Почта (`ru.yandex.mail`)

Почтовые клиенты требуют фоновой синхронизации, но Яндекс Почта использует постоянное соединение для сбора телеметрии.

Если вы готовы отказаться от мгновенных push-уведомлений и получать письма только при открытии приложения, примените жесткие ограничения.

## Рекомендуемые ограничения AppOps

### 1. Остановка фоновой синхронизации

Полностью останавливает сбор метаданных в фоне и TCP-пинги.

```sh
adb shell cmd appops set ru.yandex.mail RUN_IN_BACKGROUND ignore
adb shell cmd appops set ru.yandex.mail WAKE_LOCK ignore
```

### 2. Защита контактов

Запрещает клиенту чтение вашей телефонной книги.

```sh
adb shell cmd appops set ru.yandex.mail READ_CONTACTS ignore
```
